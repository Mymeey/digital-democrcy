# Assets

このフォルダにはアプリのアイコンとスプラッシュ画像を配置してください。

必要なファイル:
- `icon.png` - アプリアイコン (1024x1024 推奨)
- `splash.png` - スプラッシュ画面 (1284x2778 推奨)
- `adaptive-icon.png` - Android用アダプティブアイコン (1024x1024 推奨)
- `favicon.png` - Web用ファビコン (48x48 推奨)

テスト用のプレースホルダー画像を使用する場合は、以下のコマンドで生成できます:
```bash
npx expo install expo-asset
```
